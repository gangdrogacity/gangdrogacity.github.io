FAQ - Domande Frequenti

Forge è il modloader necessario per eseguire il nostro modpack. Ecco i passaggi per installarlo:

  1. Scarica l'installer di Forge 1.20.1-47.3.33 da questo link ufficiale
  2. Assicurati di avere Java 17 installato (vedi FAQ su Java)
  3. Esegui l'installer con un doppio clic o usando il comando java -jar forge-1.20.1-47.3.33-installer.jar
  4. Seleziona "Install client" e conferma
  5. Attendi il completamento dell'installazione
Nota: Forge creerà un nuovo profilo nel launcher di Minecraft chiamato "forge-1.20.1-47.3.33"

Minecraft 1.20.1 e Forge richiedono Java 17 o superiore. Ecco come installarlo:

Windows:
  1. Scarica il JDK da Adoptium (Eclipse Temurin)
  2. Seleziona la versione Windows .msi per il tuo sistema (x64 per la maggior parte dei PC moderni)
  3. Esegui l'installer e segui le istruzioni
macOS:
  1. Scarica il JDK da Adoptium
  2. Seleziona la versione macOS .pkg
  3. Esegui l'installer e segui le istruzioni
Linux:
  1. Usa il gestore pacchetti della tua distribuzione (ad esempio: sudo apt install openjdk-17-jdk per Ubuntu/Debian)
  2. In alternativa, scarica la versione Linux da Adoptium
Verifica l'installazione: Dopo l'installazione, apri un terminale o prompt dei comandi e digita java -version. Dovresti vedere una versione 17.x.x.

Un modpack è una collezione curata di mod (modifiche) per Minecraft, configurate per funzionare insieme senza conflitti.

Caratteristiche del nostro modpack:
  • Raccolta di mod selezionate per migliorare l'esperienza di gioco
  • Configurazioni ottimizzate per prestazioni e stabilità
  • Compatibilità testata tra tutte le mod incluse
  • Aggiornamenti regolari per supportare nuove mod e correzioni

Il GDC Modpack include mod per:

  • Miglioramenti grafici e di performance
  • Nuovi contenuti e meccaniche di gioco
  • Miglioramenti dell'interfaccia utente
  • Ottimizzazioni tecniche
Nota: Utilizziamo diversi formati di distribuzione (.mrpack per Modrinth e Client Legacy per installazione manuale) per adattarci alle diverse esigenze degli utenti.

Il gioco crasha all'avvio:
  • Memoria insufficiente: Aumenta la RAM allocata a Minecraft (consigliato almeno 4GB)
    • In Minecraft Launcher: Modifica il profilo Forge → Altre opzioni → Argomenti JVM → Cerca -Xmx2G e cambia in -Xmx4G (per 4GB)
    • In Prism Launcher/Modrinth App: Modifica istanza → Impostazioni → Memoria
  • Java sbagliato: Assicurati di usare Java 17
  • Versione di Forge errata: Usa esattamente la versione 1.20.1-47.3.33
  • Mod incompatibili: Se hai aggiunto altre mod, prova a rimuoverle
Problemi grafici:
  • Schermate nere: Prova a disabilitare le shader nella cartella options
  • FPS bassi: Riduci la distanza di rendering o disabilita alcune opzioni grafiche
  • Glitch visivi: Aggiorna i driver della tua scheda grafica
Multiplayer:
  • Non riesco a connettermi: Assicurati che client e server usino esattamente le stesse mod e versioni
  • Latenza elevata: Verifica la tua connessione internet e prova a chiudere altre applicazioni che utilizzano la rete
Serve aiuto? Puoi chiedere supporto sul nostro Discord o aprire una issue su GitHub.

Per un'esperienza di gioco ottimale con il GDC Modpack, consigliamo:

Requisiti minimi:
  • CPU: Intel Core i5 di 7ª generazione o AMD Ryzen 3 (o equivalente)
  • RAM: 8GB (con almeno 4GB allocati a Minecraft)
  • GPU: NVIDIA GeForce GTX 960 o AMD Radeon RX 560 o Intel Iris Xe
  • Spazio su disco: 2GB liberi
  • Sistema operativo: Windows 10/11, macOS 10.15+, o Linux con ambiente grafico
  • Java: Java 17 o superiore
Configurazione consigliata:
  • CPU: Intel Core i7 di 10ª generazione o AMD Ryzen 5 5000 series
  • RAM: 16GB (con 6-8GB allocati a Minecraft)
  • GPU: NVIDIA GeForce GTX 1660 o AMD Radeon RX 5600 XT o superiore
  • Spazio su disco: SSD con 5GB liberi
  • Connessione Internet: Stabile per gli aggiornamenti e il multiplayer
Nota sulla memoria: Allocare troppa RAM può causare problemi di performance a causa della garbage collection di Java. Se noti "stuttering" periodico, prova a ridurre la RAM allocata.
