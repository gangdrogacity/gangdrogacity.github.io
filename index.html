Tutorial per Modrinth
  1. Scarica il file .mrpack dalla sezione Releases
  2. Installa Prism Launcher o Modrinth App
  3. Aggiungi il tuo account Minecraft al launcher
  4. Importa il file .mrpack scaricato:
    • Prism Launcher: Clicca "Aggiungi Istanza" → "Importa" → Seleziona il file .mrpack
    • Modrinth App: Clicca "+" → "Importa" → Seleziona il file .mrpack
  5. Attendi il completamento dell'installazione
  6. Avvia il gioco!

Tutorial per Client Legacy
  1. Scarica il file Client Legacy dalla sezione Releases
  2. Assicurati di aver già installato Forge 1.20.1-47.3.33 (vedi FAQ sopra)
  3. Estrai il contenuto dello ZIP nella cartella .minecraft:
    • Windows: %appdata%\.minecraft
    • macOS: ~/Library/Application Support/minecraft
    • Linux: ~/.minecraft
  4. Avvia il launcher di Minecraft
  5. Seleziona il profilo Forge 1.20.1-47.3.33
  6. Avvia il gioco!
